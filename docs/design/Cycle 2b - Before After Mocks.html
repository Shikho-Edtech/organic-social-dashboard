Closing · consistency matrix

Patterns that repeat across all four pages

Pattern 1
Inline metric pills
Numbers in prose get promoted to 142k or 38%. Eye lands on signal.
Pattern 2
Benchmark overlays
Every primary chart shows a reference line: prior period, industry benchmark, or all-post baseline. No number floats alone.
Pattern 3
One viz is primary
Each page has one hero viz at full width; everything else folds beneath it. No more side-by-side charts of equal weight.
Pattern 4
Glance → scan → dig
3-tier depth on every page: Hero = 1 number. Secondary = 4–6 items. Detail = list/chart/table.
